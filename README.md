# data-modern-and-data-stack
# Modern Data Stack – Pipeline de Dados

Este projeto implementa uma arquitetura moderna de dados (Modern Data Stack) utilizando ferramentas amplamente adotadas no mercado, como **Airflow, dbt, Python e Docker**, permitindo orquestração, transformação e padronização de pipelines de dados.

---

## Tecnologias Utilizadas

### **Orquestração**
- **Apache Airflow**  
  Usado para executar, agendar e monitorar pipelines de forma escalável.

### **Transformação de Dados**
- **dbt (Data Build Tool)**  
  Responsável por transformar dados no modelo ELT (Extract → Load → Transform).

### **Infraestrutura**
- **Docker / Docker Compose**  
  Facilita a criação de ambientes reprodutíveis e portáveis.

### **Linguagem Principal**
- **Python 3.11+**
